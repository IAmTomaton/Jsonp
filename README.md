# Jsonp

Работает всё точно также как и в учебнике.
Для 4200 сервера нужно будет установить mongodb (npm install mongodb)
Запускаем оба сервера и переходим на страницу http://localhost:3000/
3000 получает данные с 4200 сервера.
Если в базе с практики есть запись {name: "Adam", serverid: 13, rank: 1},
то по нажатию кнопки customer можно будет её получить м 4200 сервера.
